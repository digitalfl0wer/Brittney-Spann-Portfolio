About Me

Hi, I’m Brie—an AI engineer and Web3 developer who turns ideas into usable, beautiful products. I started as a jewelry founder, teaching myself to build websites, tell stories online, and craft customer journeys that actually convert and feel good.

Now I work with React, Next.js, MCPs, and on-chain tools to build AI-powered, interactive experiences. I care about clear impact: more access, less friction, and more joy for the communities I build for.

Creations

Black Magic Slot Machine

Black Magic Reels is my love letter to Black joy in game form, a small-batch, Afro-chic slot machine where strict, deterministic logic runs underneath reels wrapped in rich cultural symbols, lush color, and ritual-level drama. Every spin feels like ceremony: clear math, glowing paylines, and a UI that is representative of the culture that inspired it.

Hello Hands

HelloHands is a small, loving ASL practice app I built for my family, especially my 3-year-old with apraxia to make daily signing gentle, joyful, and repeatable. This app is created with MediaPipe and Goose CLI

  • GitHub Repo

Sacred Styles

Sacred Styles is an AI style studio honoring Black hair—built by and for us. I trained the model on my own face to set a standard for consent and representation, so you can preview protective styles and colorways with texture, culture, and creativity at the core.
